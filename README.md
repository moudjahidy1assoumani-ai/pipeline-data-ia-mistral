# 🚀 Pipeline Data & IA en Production (API REST & Streamlit)

Ce projet est une application web décisionnelle (Dashboard) qui illustre un pipeline complet de traitement de données et d'interconnexion avec l'Intelligence Artificielle Générative.

## 🛠️ Fonctionnalités du Pipeline
1. **Data Engineering (Scraping) :** Extraction automatisée de contenus textuels en temps réel depuis Wikipédia via `requests` et `BeautifulSoup4`.
2. **Data Analytics (NLP) :** Implémentation d'un algorithme déterministe de notation lexicale pour l'analyse de tonalité en français.
3. **Génie IA (LLM) :** Interconnexion directe via requêtes HTTP POST (API REST) avec le modèle `open-mistral-7b` de Mistral AI pour générer des synthèses automatiques.
4. **Data Viz :** Restitution graphique dynamique des indicateurs clés (KPI) sur une interface interactive `Streamlit` avec `Matplotlib`.

## 📦 Stack Technique
- Python 3.14
- Streamlit (Interface Web)
- Requests / BeautifulSoup4 (Scraping & API REST)
- Matplotlib (Data Visualisation)
